# 🖥️ Server Monitor - سیستم مانیتورینگ سرور

یک سیستم مانیتورینگ حرفه‌ای برای نظارت بر سرور و سیستم‌های کامپیوتری با قابلیت‌های پیشرفته.

## ✨ قابلیت‌ها

### 📊 مانیتورینگ پایه
- مصرف CPU و RAM لحظه‌ای
- وضعیت دیسک‌ها
- اطلاعات شبکه
- پردازنده‌های پرمصرف
- آپتایم سیستم

### 🌡️ قابلیت‌های پیشرفته
- **مانیتورینگ دما**: CPU، GPU و قطعات مختلف
- **مانیتورینگ اینترنت**: پینگ، سرعت، اتصال
- **مانیتورینگ باتری**: درصد، وضعیت شارژ، زمان باقیمانده
- **مانیتورینگ پورت‌ها**: پورت‌های باز و اتصالات شبکه

### 🔔 هشدارها
- هشدار مصرف بالای CPU/RAM/Disk
- هشدار دمای بالا
- هشدار قطعی اینترنت یا پینگ بالا
- هشدار باتری ضعیف
- هشدار پورت‌های حساس باز
- ارسال خودکار هشدار به تلگرام

### 📈 داشبورد وب
- نمایش لحظه‌ای همه متریک‌ها
- نمودارهای تعاملی
- تاریخچه هشدارها
- آپدیت خودکار هر 5 ثانیه
- طراحی مدرن و ریسپانسیو

## 🛠 تکنولوژی‌ها

- **Python 3.11+**
- **Flask** - وب سرور
- **psutil** - دریافت اطلاعات سیستم
- **Plotly** - نمودارهای تعاملی
- **WMI** - دمای قطعات (ویندوز)
- **ping3** - پینگ
- **speedtest-cli** - تست سرعت اینترنت
